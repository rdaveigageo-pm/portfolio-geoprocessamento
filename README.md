# Urban GIS Lab

GIS-based spatial analysis applied to urban and environmental planning.

---

## 📌 About

This repository presents geospatial analysis projects developed using QGIS, focusing on urban planning, environmental assessment and territorial analysis.  

The projects are structured to demonstrate technical and analytical capabilities in geoprocessing, integrating spatial data, methodological procedures and critical interpretation.

---

## 🎯 Objectives

- Apply geotechnologies to urban and environmental analysis  
- Support territorial planning and decision-making processes  
- Develop reproducible spatial analysis workflows  
- Integrate academic research with applied GIS solutions  

---

## 🛠️ Tools and Methods

**Software:**
- QGIS

**Techniques:**
- Vector and raster geoprocessing  
- Buffer and overlay analysis  
- Spatial data integration  
- Thematic cartography  
- Multi-criteria analysis (MCA)
